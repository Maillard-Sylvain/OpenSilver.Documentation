Enum AutoCompleteFilterMode

Specifies how text in the text box portion of the AutoCompleteBox control is used to filter items specified by the AutoCompleteBox.ItemsSource property for display in the drop-down.

Namespace: Windows.UI.Xaml.Controls
Assembly: OpenSilver.Runtime.dll
