Examples

You can add a Canvas to the XAML as follows:

Or in C#:

Canvas myCanvas = new Canvas();
myCanvas.Width = 100;
myCanvas.Height = 100;
myCanvas.Background = new SolidColorBrush(Windows.UI.Colors.Blue);
myCanvas.HorizontalAlignment=HorizontalAlignment.Left;
